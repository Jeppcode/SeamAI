For future user
##################################################################################

pose_data_collection.py is the v2 live collection script. For each pedestrian it
records a trajectory (center + bounding box), a full COCO-17 skeleton, and an
mp4; it pre-sorts clips into enter/pass/exit/removed and writes per-clip and
per-session metadata. By default the dataset is written to ../data/.

It does NOT compute the handcrafted features -- that stays a separate step.

To run:
  Set the options in main() at the bottom of pose_data_collection.py, then run it
  (press Run in your editor, or `python pose_data_collection.py`). No command-line
  arguments are needed. Set save_clips=False in main() to track/show without
  saving (monitor mode).

See README.md here for the schema and the settings.
