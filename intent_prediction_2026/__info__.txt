This folder is the v2 (pose-based) pipeline added by Jesper Malmgren's master's
thesis (2026), kept separate from the original automatic_door/automatic_labeling/ scripts so
nothing existing is touched.

Headline change vs. the original data_collection.py (Norberg):
  * a single YOLO pose model (yolo11n-pose) produces the bounding box AND the
    17 COCO body keypoints in one pass -- no separate MediaPipe step;
  * a third class, "exit", is detected and labeled at collection time.

Subfolders:
  data_collection/   - the live collection script. Start here.
  data/              - where the collection script writes the dataset.
  feature_extraction/   - builds the handcrafted feature table (CSV) from the data.

See README.md in this folder and in data_collection/ for the details, the JSON
schema.
