# baseline_logreg

Logistic Regression baseline on handcrafted trajectory features.

## Contents
- `train_logreg.py` — trains/evaluates the model across TTE and observation windows
- `results/` — plots and CSV outputs (created when you run the script; not shipped)

## Why this baseline
- Fast and interpretable
- Provides an early signal of which feature families are useful

## Run
```bash
python 1_RQ1_feature_importance/baseline_logreg/train_logreg.py
```
